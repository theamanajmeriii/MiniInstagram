# MiniInstagram
A Mini Instagram-style social posts app built with Node.js, Express, and EJS where users can create and view posts through server-side rendered pages. This project was developed as a practice application to understand routing, MVC structure, and backend development with Express.   app  Tech: Node.js • Express • EJS • JavaScript • RESTful Routes
